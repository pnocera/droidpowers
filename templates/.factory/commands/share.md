# Share Command

## Usage
```bash
/droid sharing-skills
```
or
```bash
/share
```

## Description
Guides the complete process of contributing skills back to the upstream repository through proper git workflow, pull request creation, and community engagement.

## Process
1. **Preparation**: Verify skill quality, test with subagents, ensure compatibility
2. **Repository Operations**: Create feature branch, commit with proper messages, push to remote
3. **Pull Request**: Create comprehensive PR, fill contribution forms, address feedback
4. **Community Engagement**: Professional communication, responsive collaboration

## Examples
```bash
/droid sharing-skills
# Guides through complete upstream contribution workflow
# Ensures proper git operations and PR creation
# Includes quality standards and best practices

/share
# Shortcut command for sharing-skills droid
# Same comprehensive contribution guidance
```

$ARGUMENTS

## Required Tools
- Git for version control and branching
- GitHub CLI for streamlined PR creation
- Testing-skills-with-subagents for validation
- Community guidelines for standards compliance

## Quality Gates
- [ ] Skill tested with testing-skills-with-subagents
- [ ] Documentation complete and clear
- [ ] Follows project conventions

- [ ] Community benefit demonstrated
