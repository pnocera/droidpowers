# Using Droids Command

## Usage
```bash
/droid using-droids
```

## Description
Establishes mandatory workflows for discovering and using droids properly.

## Process
1. Check if any droid applies to the current request
2. Use skill-checker for automatic task analysis
3. Read applicable droids completely
4. Create TodoWrite todos for checklist items
5. Announce which droid(s) you're using
6. Follow droid procedures exactly

```bash
# Provides guidance on proper droid discovery and usage
```
